NAME
        cp.copy - recursive copy used in cp

DESCRIPTION
        This utility is the backend of the cp(1) utility.

USAGE
        local copy = require("cp.copy")

        copy(files:table) or copy(...:strings)
          -- copies files recursively in the same manner as cp(1).
