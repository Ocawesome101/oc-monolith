NAME
        umount - unmount filesystems

SYNOPSIS
        umount PATH

DESCRIPTION
        The umount command detaches the mentioned filesystem from the file hierarchy.  A filesystem is specified by giving the directory where it has been mounted.

EXAMPLES
        umount /mnt/ca3
            Unmount the filesystem mounted at /mnt/ca3.
